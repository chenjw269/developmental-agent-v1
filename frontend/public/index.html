智能体 API 网关演示系统

本页为前端占位，用于 Docker Compose 连通性验证。实际业务请使用 React 前端对接下方接口。

服务入口

  • 管理后端 API（申请 / 审批 / 配额）: http://localhost:8000/docs
  • APISIX 代理（需 API Key）: http://localhost:9080/agent/chat（POST）
  • 模拟智能体直连（仅演示）: http://localhost:8001/health

验证步骤

  1. 在管理后端提交申请 → POST /api/v1/applications
  2. 管理员审批通过 → POST /api/v1/applications/{id}/approve，获得 API Key
  3. 使用 API Key 调用网关: curl -X POST http://localhost:9080/agent/chat -H "X-API-KEY: <key>" -H "Content-Type: application/json" -d "{\"message\":\"hi\"}"
